# 🎬 Toy Story OVA: Anime Showdown
### *"When Toys Meet Anime Figures"*

An original fan fiction story where **Woody, Buzz, and Bonnie's toys** meet **six legendary anime action figures** — Goku, Naruto, Luffy, Ichigo, Deku, and Tanjiro — in the most unexpected crossover playdate of all time.

---

## 📖 Story Structure

| Act | Chapters | Theme |
|-----|----------|-------|
| **Act I — The Arrival** | Ch.1 "A Quiet Morning in Bonnie's Room" · Ch.2 "Six Warriors Walk Into a Bedroom" | Setup: The worlds collide |
| **Act II — The Showdown** | Ch.3 "Whose Room Is This Anyway?" · Ch.4 "The Cracks in the Armor" | Conflict: Truth surfaces |
| **Act III — The Unity** | Ch.5 "Operation: Get Back" · Ch.6 "Until Next Time, Cowboys" | Resolution: New purpose |

## 🛠️ Tech Stack

Pure **HTML5 + CSS3 + Vanilla JavaScript** — no frameworks. Designed for GitHub Pages.

- Semantic HTML with accessible markup
- Character-colored dialogue system
- Dark mode / light mode toggle
- Reading progress bar
- Responsive design (desktop, tablet, mobile)
- Google Fonts: Fredoka One + Nunito + Baloo 2
